=====================
Python/ASAP Tutorials
=====================


This document contains the tutorials for the 2009 ATNF Radio School at Parkes.
These are in order of difficulty. The first one is a very basic introduction into python.

These tutorials show how to do data redcution of single-dish spectral-line pointed obeservations.
It doesn't address interferometry, continuum observation or single-dish mapping data.
