ASAP background

ASAP is developed in c++ using a thin python front-end, to make it scriptable. The ASAP user interface is ipython. The c++ code reuses several casa libraries. It has been developed in the last nine months and has been released to ATNF users. ASAP v2.0 underwent a major re-design to support varying numbers of channels per IF (or Beam). It also allows much more flexible data selection via the seletcor object. Currently, the package has only been tested on linux (debian) system, other linux and OSX binaries are going to be avaliable soon.
